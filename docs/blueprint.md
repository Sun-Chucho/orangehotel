# **App Name**: OrangeFlow

## Core Features:

- Role-Based Dashboards: Secure login with distinct dashboards for Hotel Manager, Food & Inventory Manager, Cashiers, Kitchen POS, and Barista POS, each tailored to their specific responsibilities.
- Real-Time Monitoring: Managers can monitor total daily revenue, room occupancy, sales summaries, and staff activity in real time.
- Inventory Management: Track stock levels, purchases, usage, and supplier history for food, bar items, and general supplies.
- POS Systems: Dedicated POS interfaces for Kitchen and Barista roles, supporting order creation, modifiers, table/room charges, and order status tracking.
- Automated Reporting: Generate daily sales, shift, stock, and profit & loss reports with export options for PDF and Excel.
- AI Sales Trend Forecast: Uses AI to predict future sales trends based on historical data.
- User-Friendly Interface: Responsive design, fast POS interface, and offline mode for uninterrupted operation.

## Style Guidelines:

- Primary color: Orange (#F57C00) for primary interactive elements and highlights.
- Background color: Off-white (#F2F2F2), subtly tinted towards orange, for a clean content background.
- Accent color: Dark orange (#D96A00) for call-to-action elements, providing clear visual contrast.
- Body and headline font: 'Inter', a sans-serif font that delivers a clean and modern look; great for both headlines and body text.
- Left vertical black sidebar with an orange active tab highlight for clear navigation. Top navigation bar includes user info and notifications.
- Use a set of consistent icons throughout the UI. These should use a 'thin' or 'outline' style to reduce visual clutter.
- Subtle transitions and animations to acknowledge user interactions (e.g., button presses, data updates).
